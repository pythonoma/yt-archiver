# yt-archiver

Install:
```
curl -L https://github.com/pythonoma/yt-archiver/raw/master/yt-archiver.py -o /usr/bin/yt-archiver.py
curl -L https://github.com/pythonoma/yt-archiver/raw/master/archive_uploader.py -o /usr/bin/archive_uploader.py
curl -L https://github.com/pythonoma/yt-archiver/raw/master/yt_downloader.py -o /usr/bin/yt_downloader.py

chmod a+rx /usr/bin/yt-archiver.py

yt-archiver.py
```

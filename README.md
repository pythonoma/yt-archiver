# yt-archiver

VPS intial setup:
```
curl -L https://raw.githubusercontent.com/pythonoma/yt-archiver/master/install.sh -o install.sh
chmod a+rx install.sh
./install.sh
```
===========================================

Configure archive.org account:
```ia configure```

===========================================

Run:
```
yt-archiver.py -u <Youtube URL> -i <Archive_Page_Identifier>
```
